# 🍽️ Plateful – Food Ordering Website

Plateful is a full-stack food ordering web application built using **Python** and **Django**. The platform allows users to browse delicious meals, add items to their cart, place food orders, and enjoy a seamless online food ordering experience through a clean and user-friendly interface.

## 🚀 Live Demo

🌐 **Website:** https://plateful-idf5.onrender.com

---

## 📌 Features

* 🍕 Browse a variety of food items
* 🔍 Search and explore available dishes
* 🛒 Add and remove items from the shopping cart
* 👤 User Registration & Login Authentication
* 📦 Place food orders
* 📱 Responsive and user-friendly interface
* 🔒 Secure authentication using Django's built-in authentication system
* 🗂️ Admin panel for managing food items and users
* 💾 Database integration for storing user and order information

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/plateful.git
```

### 2. Navigate to the Project

```bash
cd plateful
```

### 3. Create a Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Apply Migrations

```bash
python manage.py migrate
```

### 6. Run the Server

```bash
python manage.py runserver
```

Now open:

```text
http://127.0.0.1:8000/
```
